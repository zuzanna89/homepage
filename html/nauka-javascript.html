JavaScript

Do czego JavaScript

Pozwala na interakcje z użytkownikiem i "robi rzeczy". Korzysta ze skryptu ECMA (ES6+), który jest ustandaryzowanym językiem programowania. Kod wrzucamy tuż przed zakończeniemy body, zaczynamy go znacznikiem script. Aby połączyć kod z dokumentem js należy użyć opcji src (nie href).

JavaScript składa się z:

-wartości,
-operacji porównywania,
-literałów,
-zmiennych,
-obiektów
-i funkcji.

Polecenie console.log pozwala na zobaczenie działania kodu w konsoli (console), która znajduje się w narzędziach deweloperskich. Alt+command+I pozwala otworzyć narzędzia deweloperskie. W Windowsie i w Linuksie są to F12.

const (zmienna) pozwala przypisać wartości np. const age, gdzie w osobnym miejscu zdefiniujemy nasz wiek age=32, lub const firstName = 'Zuzia', ma stałe połączenie. Zmienna let pozwala zmienić (podmienić) wartość np. let newName i np. pod newName dopisujemy inne imię.
literały są zapisywane przez tzw. backtick `` i wyglądają tak: console.log ('Siema, nazywam się ${firstName')i mam ${age} lat.`)

Typy proste w js (primitives)

string - tekst, który musimy wziąć w pojedynczy lub podwójny cudzysłów.
number
bigint - duża liczba
boolean - true lub false
wartość celowo niezdefiniowana, po to, abyśmy mogli ją później dopisać
undefined - brak definicji
symbol (rzadko używane)
polecenie typeof pozwala sprawdzić jakiego typu jest dana zmienna np. typeof newName=string, typeof age=number, typeof carColor=null.
Immutability oznacza, że nie można zmienić primitive values, możemy jedynie dopisać im wartości, a raczej edytować połączenie między nimi.

Funkcje w JS

Funkcje posiadają argumenty, np.liczby, a słowem kluczowym jest return, które determinuje co ma nam zwrócić dana funkcja - jakiego rodzaju wynik. Jeśli chcemy uzyskać wynik w rozumieniu matematycznym, możemy użyć nazwy funkcji calculate. Jeśli funkcja nie ma zdefiniowanego return, to będzie to undefined.
Argumentów możemy podać kilka w jednym miejscu lub nie podawać żadnego.

Po co stosuje się funkcje?

Deklarujemy kod, który się powtarza zgodnie z zasadą DRY (don't repeat yourself).
Kod staje się też bardziej czytelny.
Łatwiej nam znaleźć błędy.
Wystarczy, że zmienimy dane w jednym miejscu, by zaktualizowały się dane wszędzie.

JavaScript jest językiem obiektowym

Nasz świat jest obiektowy - np. człowiek ma imię, kolor skóry, wiek, kolor oczu, włosów, zawód itp.
Samochód ma kolor, moc silnika, liczbę koni mechanicznych, liczbę kół, liczbę miejsc itp., są to tak zwane właściwości lub własności (properties).

Zarówno człowiek, jak i samochód mogą wykonywać różne akcje (zadania), np. człowiek może chodzić, jeździć, mówić, tańczyć, spać a samochód jeździć do przodu, cofać się, używać klaksona, wycieraczek itp.

Również obiekty w JavaScript możemy opisać.
Używamy do tego par, które składają się z klucza i wartości (key:value), np. diameter:10000, name:'deathStar' i na końcu wstawiamy , .

Logika w JS

W JavaSrcipcie występuje kilka podstawowych operatorów (porównań), które pozwalają nam na zdefiniowanie, czy dane porównanie to prawda, czy fałsz. W logice prawda i fałsz są podstawowymi wartościami i podobnie funkcjonuje komputer oraz programowanie. Jednym ze słów kluczowych jest "if", które w zależności od tego jaki ustalimy operator w wyrażeniu logicznym, będzie nam zwracać określone wyniki.

  • > - lewa strona większa od prawej
  • < - lewa strona mniejsza od prawej
  • >= - lewa strona większa/ równa prawej
  • <= - lewa strona mniejsza/ równa prawej
  • == - obie strony równe
  • === - obie strony równe (sprawdza też typeof)
  • != - obie strony różne
  • !== - obie strony różne (sprawdza też typeof)

Wartości falsy:
  • false
  • 0
  • ","",`` (puste string
  • null

Wartości truthy:
  • true

  • {} (obiekty)

Oczywiście istnieją także inne wyrażenia:
  • AND - wszystkie warunki muszą być spełnione (if..and - &&)
  • OR - przynajmniej 1 argument musi być prawdą
  • NOT - argument odwrócony (if!() - używać z rozwagą (!false=>true, !true=false)
  • IF..ELSE - to taka wartość jak w Excelu, jeśli (prawda), jeśli (fałsz), można dodać kilka else, wtedy kod będzie szukał prawdy i pokaże pierwszy wynik, który jest prawdziwy.
  • SWITCH - switch, case 1:, break; case:2: break; default:, switch pozwala na wpisanie kilku przypadków i posiada opcję default, która definiuje jakie jest domyślne wyrażenie.

Eventy

Jak wiemy, programowanie polega na tłumaczeniu języka naturalnego na język zrozumiały przez komputer. Od JS oczekujemy, że zamieni słowa na czyny:) Button to znacznik w html-u, który w przeciwieństwie do <a>, który odsyła nas w jakieś miejsce, button połączony z js pozwala na interakcje na stronie. Może też być wykorzystany zamiast input i submit do przesyłania formularzy. Pozwoli nam także na rozwinięcie menu np. podczas tworzenia menu typu hamburger. Za pomocą event w js możemy wykonać jakąś akcję. Częste są np. akcje mouseEvent (np. click) z funkcją addEventlistener, które wykonują jakieś zadanie po kliknięciu myszką, np. zamiana tekstu nagłówka po kliknięciu myszką czy rozwinięcie hamburger menu. W poniższym przykładzie hamburger menu zgodnie z ustawieniami media query jest widoczny jedynie na urządzeniach mobilnych. Importowanie modułów JS


Import i export są stosowane, aby nie przeciążać kodu. W przeciwnym razie byłby on bardzo długi i rozbudowany. Pozwalają one także na korzystanie z fragmentów kodu innych deweloperów lub dzielenie się własnym czy utrzymanie porządku. Common JS występuje w skryptach node'owych, a ES6 w przeglądarce. Żeby móc wykonać import potrzebujemy komendy <script type="module"> Aby podpiąć import pod dany skrypt, tworzymy nowy plik js, w którym umieszczamy skrypt i używamy polecenia export dla danych funkcji. W pliku docelowym, do którego ma zostać zaimportowany skrypt, wpisujemy natomiast import podając nazwę konkretnej funkcji i from podając źródło. W przypadku importu nazwanego stosujemy nazwę funkcji w nawiasie sześciennym, jeśli funkcja jest defaultowa, nie używamy nawiasu. Można też zastosować skrócone nazwy. Możemy wtedy zastosować komendę import długaistniejącanazwa as druga własnakrótkanazwa. Użycie gwiazdki as pozwala pobranie/importowanie obiektu.
